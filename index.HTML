FreeGhost

Projeto Carrinho Arduino

Equipe FreeGhost

Este trabalho teve dois principais objetivos: primeiro, conectar a
linguagem de programação com aplicações do mundo real através do
Arduino; segundo, criar um carrinho capaz de operar em ambientes
inacessíveis aos humanos, como locais com resíduos perigosos. Antes de
iniciar o projeto, houve um aprendizado em lógica de programação e
todos os requisitos foram abordados. Ao final, o resultado foi um
carrinho multifuncional, capaz de se mover para frente, para trás,
para direita, para esquerda e em alta velocidade.
